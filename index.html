HeimisheMaps Digital Product Manager

A private shop automation tool built on the Etsy Open API v3 — for internal use with the HeimisheMaps store only.

📋

Bulk Listing Creation

Automates structured listing drafts for new product lines — titles, descriptions, tags, categorization — then submits via the Listings API. Eliminates copy-paste errors across 50+ SKUs.

🔄

Listing Updates & Sync

Applies bulk description updates, price adjustments, and tag refreshes via the Listings PATCH endpoint. Used for seasonal promotions, SEO updates, and catalog-wide formatting.

🔍

SEO Tag Optimization

Analyzes Etsy search trends and updates listing tags across the catalog to improve discoverability. Changes apply exclusively to HeimisheMaps-owned listings.

💰

Pricing Management

Supports dynamic pricing across listing groups — promotional discounts, seasonal adjustments, and cost-based reviews — applied only to active HeimisheMaps listings.

📦

Order Fulfillment Tracking

Reads recent orders to monitor digital download fulfillment status and flags any requiring manual attention, such as buyer non-delivery reports.

📊

Shop Analytics

Reads listing-level stats — views, favorites, revenue — to inform restocking priorities and surface top-performing products via read-only analytics endpoints.

Technical Details

Node.js backend service running on private infrastructure. Authenticates with Etsy using OAuth 2.0 + PKCE (Etsy v3 required). All credentials are owner-only — no third-party access, no scraping, no buyer data storage. Rate-limited per Etsy's published guidelines.

It is a private single-shop tool — not a multi-tenant SaaS — with no external users or commercial licensing.
